Detailed Description

An abstract, base class representing a mathematical expression.

The Expression class is an abstract class that defines regular and virtual functions used to perform various mathematical operations. These include basic number and variable expressions, as well as operational expressions such as addition, multiplication, and let substitution. All classes that inherit from the Expression class are also able to print as a string in two different styles.
