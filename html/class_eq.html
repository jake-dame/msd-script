Detailed Description

An Expr derived class representing an equality operation/comparison.

The Eq class compares two other Expr objects, and checks for equality. Its value is defined by a BoolVal object. E.g. "Expr == Expr". If the Expr objects are equal, an Eq object will have a BoolVal of "_true". If they are not, it will be "_false."

Replaces a Var within an Eq object with another Expr object.

Parameters
strThe string value, or "variable", to replace
eThe Expr object to replace the "variable" with
Returns
A new Eq object, with the necessary substitutions

If an Eq contains a user-inputted string, the string will be replaced where it occurs with another user-inputted parameter. The Var class is responsible for checking whether the string that is searched for is actually contained by the Expression calling it. (See: Var::subst() ). This method is called recursively on both the lhs and rhs values of an Eq object. The "variable" in question is replaced at all levels of nesting.
